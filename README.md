# 📚 Personal Book Manager

A full-stack Personal Book Manager built with the MERN stack and Next.js.

Users can create an account, log in securely, and manage their personal collection of books.

---

## 🚀 Features

### Authentication
- User Signup
- User Login
- JWT Authentication
- Protected Routes

### Book Management
Users can:

- Add books
- Edit book status
- Delete books
- Track reading progress

Book Status:
- 📖 Want to Read
- 📘 Reading
- ✅ Completed

### Dashboard
Displays:

- Total books
- Books currently reading
- Completed books
- List of books with status

---

## 🛠 Tech Stack

Frontend:
- Next.js
- Tailwind CSS
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Authentication:
- JWT (JSON Web Tokens)

Deployment:
- Vercel (Frontend)
